PART 1: MINDSET DOCUMENTATION (README.md lines 247-272)
================================================================================

employment.py module calculates employment changes from output changes using:

INPUT:
- EMPL_COEF: Country- and sector-specific employment-output elasticities
- empl_base: Employment baseline from LABOR_BASE
- q_base: Baseline output from Leontief calculation
- dq: Output changes

OUTPUT:
- empl_multiplier: Employment elasticities × employment intensity (people per kUSD)
- dempl: Employment changes (absolute)

FORMULA (from documentation):
  empl_multiplier = EMPL_COEF × (empl_base / q_base)
  dempl = empl_multiplier × dq

B. employment.py (lines 36-48) - How employment is calculated
------------------------------------------------------------
    def calc_empl_multiplier(self, empl_base, q_base):
        empl_coef = self.EMPL_COEF.loc[:,"empl_coef"].to_numpy()

        self.empl_multiplier = empl_coef * (np.divide(
            empl_base, q_base, out=np.zeros_like(empl_base), where=q_base!=0))

        return True

    def calc_dempl(self, dq):
        if type(dq) != list:
            dempl = self.empl_multiplier * dq
            return dempl

✓ VERIFIED: empl_multiplier = empl_coef × (empl_base / q_base)
✓ VERIFIED: dempl = empl_multiplier × dq
✓ VERIFIED: Handles division by zero
